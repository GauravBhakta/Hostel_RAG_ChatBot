# 🤖 RAG-Based Hostel Chatbot

This project implements a chatbot using **RAG (Retrieval-Augmented Generation)**.

It understands user queries using embeddings and retrieves relevant hostel information.

---

## 🚀 Features

- Semantic search using embeddings  
- FAISS vector database  
- LLM responses using Groq  
- Streamlit web interface
